# ENERGOSTROY-WP

Промежуточная документация по WordPress-теме `energostroy`.

## Структура проекта

- `assets/css/` — стили проекта: базовые файлы, компоненты и стили отдельных страниц.
- `assets/js/` — JavaScript-логика и подключаемые скрипты интерфейса.
- `inc/` — служебные PHP-файлы темы (настройка темы, регистрация функций и возможностей WordPress).
- Шаблоны в корне темы (`front-page.php`, `footer.php`, `page-projects.php`) — разметка страниц и секций сайта.

## Требования

- Node.js
- npm

## Быстрый запуск

1. Установить зависимости:
```bash
npm install
```

2. Запустить локальную синхронизацию и автообновление:
```bash
npm run sync
```

Скрипт `sync` запускает BrowserSync через прокси `localhost/energostroy` и отслеживает изменения:
- `*.php`
- `assets/css/**/*.css`
- `assets/js/**/*.js`

## Важные CSS-файлы

- `assets/css/variables.css` — CSS-переменные проекта (цвета, размеры, шрифты и т.д.).
- `assets/css/utilities.css` — утилитарные классы и вспомогательные стили для переиспользования в шаблонах.

## Как менять контакты (шапка и подвал)

В админке WordPress:
`Внешний вид → Настроить → Контакты сайта → Телефон и Email`

После изменения полей нажмите **«Опубликовать»** — номер и почта обновятся в шапке и подвале сайта.
