# ENERGOSTROY-WP

Промежуточная документация по WordPress-теме `energostroy`.

## Структура проекта

- `assets/css/` — стили проекта: базовые файлы, компоненты и стили отдельных страниц.
- `assets/js/` — JavaScript-логика и подключаемые скрипты интерфейса.
- `inc/` — служебные PHP-файлы темы (настройка темы, регистрация функций и возможностей WordPress).
- Шаблоны в корне темы (`front-page.php`, `footer.php`, `page-projects.php`) — разметка страниц и секций сайта.

## Требования

- Node.js
- npm

## Быстрый запуск

1. Установить зависимости:
```bash
npm install
```

2. Запустить локальную синхронизацию и автообновление:
```bash
npm run sync
```

Скрипт `sync` запускает BrowserSync через прокси `localhost/energostroy` и отслеживает изменения:
- `*.php`
- `assets/css/**/*.css`
- `assets/js/**/*.js`

## Важные CSS-файлы

- `assets/css/variables.css` — CSS-переменные проекта (цвета, размеры, шрифты и т.д.).
- `assets/css/utilities.css` — утилитарные классы и вспомогательные стили для переиспользования в шаблонах.

## Как менять контакты (шапка и подвал)

В админке WordPress:
`Внешний вид → Настроить → Контакты сайта → Телефон и Email`

После изменения полей нажмите **«Опубликовать»** — номер и почта обновятся в шапке и подвале сайта.

---

## Как добавить проект

В админке WordPress:
`Проекты → Добавить проект`

Заполните **название**, **«Цитату»** (краткое описание), **«Миниатюру записи»** и основной текст проекта в редакторе.

Нажмите **«Опубликовать»** — проект появится на странице **«Проекты»**.

---

## Как добавить изображение в карточку проекта

В админке WordPress:
`Проекты → Добавить проект → Правая панель → Миниатюра записи`

Загрузите или выберите изображение из медиабиблиотеки.

После публикации оно появится в карточке проекта на странице **«Проекты»**.

---

## Как добавить краткое описание карточки

В админке WordPress:
`Проекты → Добавить проект → Цитата`

Заполните поле **«Цитата»** — этот текст выводится в карточке рядом с изображением.

Если поле пустое, сайт подставит часть текста из основного описания проекта.

---

## Как добавить описание проекта

В админке WordPress:
`Проекты → Добавить проект → Основной редактор`

Добавьте описание проекта обычными блоками WordPress.

Можно указать заказчика, сроки, место выполнения и состав работ.

---

## Как добавить галерею проекта

В админке WordPress:
`Проекты → Добавить проект → Основной редактор → + → Галерея`

Выберите изображения из медиабиблиотеки или загрузите новые.

После публикации галерея будет на детальной странице проекта по кнопке **«Подробнее»**.

---

## Как изменить проект

В админке WordPress:
`Проекты → Все проекты → Выбрать нужный проект`

После изменений нажмите **«Обновить»** — правки появятся на сайте.

---

## Как удалить проект

В админке WordPress:
`Проекты → Все проекты → Навести на проект → Удалить`

Проект попадёт в корзину и исчезнет с сайта.
