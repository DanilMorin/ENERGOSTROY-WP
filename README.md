# ENERGOSTROY-WP

Документация по WordPress-теме `energostroy` для разработчика и контент-менеджера.

## Что Это За Тема

Тема для корпоративного сайта с:
- главной страницей (`front-page.php`);
- страницей проектов (`page-projects.php`);
- детальной страницей проекта (`single-project.php`);
- отдельной страницей политики (`page-privacy.php`);
- кастомным типом записи `project`.

## Структура Проекта

- `assets/css/` — стили: базовые, компонентные, постраничные.
- `assets/js/` — клиентская логика (слайдер, меню, поведение хедера).
- `assets/img/`, `assets/fonts/` — статические ресурсы.
- `inc/` — настройка темы и интеграция с WordPress.
- Файлы шаблонов в корне темы — разметка страниц.

## Требования

- WordPress 6.x
- PHP 8.0+
- Node.js 18+
- npm 9+

## Локальный Запуск Для Разработки

1. Установить зависимости:
```bash
npm install
```

2. Запустить BrowserSync:
```bash
npm run sync
```

Скрипт `sync` использует прокси `localhost/energostroy` и отслеживает:
- `**/*.php`
- `assets/css/**/*.css`
- `assets/js/**/*.js`

## Проверка Перед Коммитом

Проверка PHP-синтаксиса:
```bash
npm run lint:php
```

## Настройка Контактов

В админке WordPress:
`Внешний вид → Настроить → Контакты сайта`

Поля:
- телефон;
- email.

После изменения нажмите **«Опубликовать»**.

## Контент-Менеджеру: Как Работать С Проектами

### Добавить проект

`Проекты → Добавить проект`

Заполните:
- заголовок;
- поле **«Цитата»** (краткое описание карточки);
- **«Миниатюра записи»**;
- основной текст в редакторе Gutenberg (включая галерею, если нужна).

Нажмите **«Опубликовать»**.

### Изменить проект

`Проекты → Все проекты → нужный проект → Обновить`

### Удалить проект

`Проекты → Все проекты → Удалить`

### Где берётся текст карточки

- если заполнена **«Цитата»** — используется она;
- если поле пустое — берётся фрагмент из основного текста проекта.

## Ограничения На Загрузку Изображений

Логика задаётся в `inc/theme-setup.php`.

- SVG: до `2 MB`.
- Остальные изображения (`image/*`): до `4 MB`.
- SVG могут загружать только пользователи с правами `manage_options` (администратор).

Если появляется ошибка загрузки SVG:
- проверьте, что вы администратор;
- убедитесь, что в SVG нет скриптов, inline-обработчиков (`onclick` и т.д.) и встроенных потенциально опасных объектов;
- уменьшите размер файла при превышении лимита.

## Технические Примечания

- Для типа записи `project` используется slug: `/projects/...`.
- В теме есть fallback-шаблон `index.php` для случаев, когда WordPress не находит более специфичный шаблон.
- Контакты в шапке и подвале берутся из Customizer (`get_theme_mod`).

## Полезные Файлы

- `inc/theme-setup.php` — поддержка темы, ограничения загрузки, SVG-валидация.
- `inc/post-types.php` — регистрация `project`.
- `inc/customizer.php` — поля контактов.
- `inc/enqueue.php` — подключение CSS/JS.
